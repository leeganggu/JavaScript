9-1 폼 요소에 접근하는 여러가지 방법
id, class 값을 사용해 폼 요소에 접근
document.querySelector 를 사용하여 접근
id는 앞에 '#'을 넣고 class는 앞에 '.'을 넣는다.
name 값을 이용하여 접근
폼 요소에 name 값을 지정하여 접글 할수 있다
지정한 name 값으로 바로 접근한다.
폼 배열을 사용하여 접근
document 속성 중 forms는 문서 안의 form 요소를 모두 불러와 배열 형태로 반환한다.
반환된 배열을 사용하여 요소에 접근하여 사용할 수 있다.
